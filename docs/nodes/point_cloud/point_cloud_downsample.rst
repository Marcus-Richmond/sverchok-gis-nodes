Point Cloud Downsample
======================

Functionality
-------------

Reduces de density of inputted point clouds.

Inputs
------

*Point Cloud*
*Every N*
*Voxel Size*
*Bounding Box*

Parameters
----------
*Method*: "Uniform", "Voxel", "Voxel and Trace"


Output
------

*Point Cloud*
*Index*
